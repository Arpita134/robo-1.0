# Robo Speaker 1.0

Robo Speaker is a simple text-to-speech (TTS) terminal app built in Python using `gTTS` and `playsound`.

You type any text, choose a language, and the script speaks it aloud.  
Type `quit` to exit.

Features

- Text-to-speech using Google's gTTS  
- Supports multiple languages  
- Simple command-line interface  
- Auto-deletes audio after playing
